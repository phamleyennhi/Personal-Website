Building my own website:

Recently, I have been working on building my personal website. It is a short term project for me to get familiar with html and css. When I first finished learning html and css on CodeAcademy last summer, I did not have much time left to start the project, but I really wanted to start it as soon as possible. Therefore, this summer when I finally have a break from college, I decided that it was the time.

In the first day of the project, I was struggling a bit to remember how html and css works. After a while, it was getting so interesting, and I dived into the work for 6 hours. I planned how the website would look like on a paper, installed Brackets to code, hosted my website using gitHub. After getting the structure of the website done, I started using css to style my website to make it look more dynamic with colors and background images.

It was a fun experience without any pressure like group projects in my computer science classes, and I now have my own website to update and connect with others.
